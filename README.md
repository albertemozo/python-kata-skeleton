## Requirements

- Python 3

## Getting Started

Create a virtual environment.
```bash
python -m venv .venv
source .venv/bin/activate
```

Install dependencies.
```bash
pip install -r requirements.txt
```

Run the tests.
```bash
pytest
```

Check types.
```bash
mypy --strict .
```
